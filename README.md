Predicting PFAS Drivers in Public Water Systems
Overview

This repository contains the code and modeling workflow used to identify industrial and geospatial drivers of PFAS detections in U.S. Public Water Systems (PWS). The project integrates geospatial feature engineering with machine learning to uncover patterns linking industrial activity to PFAS occurrence.

The primary objective is to build a predictive framework that can:

Identify regions at higher risk of PFAS contamination
Highlight key industrial sectors associated with detections
Provide interpretable insights to guide monitoring and policy decisions
